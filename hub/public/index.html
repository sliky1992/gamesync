Set up a device

  1. Download the Windows client on your Gaming PC and ROG Ally:

    The client package hasn't been uploaded to the hub yet.

    Self-contained — no .NET install needed. ~30 MB.

  2. Unzip the whole folder (e.g. into Downloads).
  3. Double-click Install-GameSync.cmd. It will:
    • ask for the Hub URL (use ) and a device name (e.g. Gaming-PC or ROG-Ally),
    • ask for Administrator (click Yes),
    • copy itself to C:\Program Files\GameSync and install + start the background service automatically.
    That's it — it now runs on every boot. (To remove later: run Uninstall-GameSync.cmd.)
  4. Map save paths: open a game in the Library, and for each device click Browse to pick its save folder. Then play — saves sync automatically.
Prefer to do it manually / set HubUrl yourself?

Edit appsettings.json next to the exe:

"HubUrl": "",
"DeviceName": "Gaming-PC"

Then either double-click GameSync.Client.exe to run it in a window, or drop a shortcut to it in shell:startup to run at login.
